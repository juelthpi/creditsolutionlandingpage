Weekly Matching Commission

The weekly matching system works similarly to the daily matching system, but there is no carry, only flash. The matching commission is calculated weekly and sent every Monday. If a child does not meet the required match amount, their sales will be reset to ₮0 (flashed) for the next cycle

Daily Return of Investment

When a user purchases a plan, they will receive a daily return on their investment, which ranges from 0.50% to 2.75% of the plan price. For example, if a user buys the Crest plan for ₮500 and their daily return is set at 1.5%, they will receive ₮7.50 per day as a return on their investment. The user will receive this return for 100 days, excluding Sundays

Withdraw System

Users can withdraw their money when the amount reaches at least ₮10. A 4% processing charge will be applied to the requested amount, and the processing time can take up to 24 hours.
